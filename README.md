## 使用说明

### 安装

进入[下载页面](../../releases/latest)下载最新的安装程序（.exe 文件）并安装。

### 开启透明模式

安装后透明模式默认开启，关闭蔷薇花园的壁纸加载（左栏 -> 其他 -> 壁纸加载）即可看到效果。

### 开启弹幕模式

你可以在窗口内或者在任务栏托盘图标上右键，然后勾选“启用弹幕模式”选项开启。
